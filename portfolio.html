Intelligent Power Saving

This app is the user interface of our building management system. Our building management system consists of a couple works such as sensing, data aggregation, data analysis, middleware and user interface. This is the Android version user interface for our system. In addition, the underlying analytic work of this system, which is based on an EM-GAM-AR model, is introduced in this page as well.




Water Flow Rate Monitoring

Since I was assigned to smart home group at Intel, I was doing this water monitoring project for the time there. This project concentrates on identify people by analyzing their water use data.

Intelligent Home Care

The Intelligent Home Care system is the final project of the Internet of Things class. Motivated by current situation in Taiwan, we use the techniques learned from the class to implement this innovative system. Since the current solutions to elders' care in Taiwan are not good enough, we focus on ameliorating the problems caused by traditional care systems.
